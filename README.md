# dia24demaio

Um projeto simples de página web criada para praticar os primeiros passos com HTML, CSS e JavaScript.

## 📌 Sobre o projeto

Este site exibe uma mensagem de boas-vindas e inclui um botão que mostra um alerta ao ser clicado. O objetivo do projeto é servir como base para aprendizado inicial de desenvolvimento web e integração com Git/GitHub.

## ✨ Funcionalidades

- Página inicial com layout centralizado
- Texto de apresentação
- Botão interativo que exibe um `alert()` ao clicar
- Estrutura organizada em arquivos separados:
  - `index.html`
  - `style.css`
  - `script.js`

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## ▶️ Como executar localmente

1. Clone o repositório:

   ```bash
   git clone https://github.com/marcosdemetriopacheco-ship-it/dia24demaio.git
   ```

2. Acesse a pasta do projeto:

   ```bash
   cd dia24demaio
   ```

3. Abra o arquivo `index.html` em seu navegador.

> Se preferir, você também pode usar uma extensão como Live Server no VS Code para visualizar a página em tempo real.
